# BERT_my_summary

预训练的语言模型的一些总结和个人思考

语言模型的发展：n-gram, CBOW/skip-gram, transformer, bert/gpt。我们不关注目前众多的语言模型变种，只关注里程碑性质的关键论文和思想。
从这些基础出发，也可以做一些发散性的思考。
